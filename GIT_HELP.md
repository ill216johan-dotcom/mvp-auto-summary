## ГЛАВНЫЕ КОМАНДЫ:

### 1️⃣ Посмотреть все ветки:
```
git branch -a
```

### 2️⃣ Переключиться на другую ветку:
```
git checkout main
```
или
```
git checkout feature/python-orchestrator
```

### 3️⃣ Скачать свежие изменения с GitHub:
```
git pull
```

### 4️⃣ Сохранить свои изменения перед переключением:
```
git add .
git commit -m "мои изменения"
```

### 5️⃣ Посмотреть где я сейчас:
```
git status
```

---

## 🚨 ЧТО ДЕЛАТЬ ЕСЛИ:

### "error: would be overwritten"
Гит ругается потому что есть несохранённые файлы.

**Решение 1** - Сохранить изменения:
```
git add .
git commit -m "сохраняю прогресс"
git checkout ДРУГАЯ_ВЕТКА
```

**Решение 2** - Забить на изменения (перезаписать):
```
git checkout -f ДРУГАЯ_ВЕТКА
```
